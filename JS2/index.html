1) Difference between var, let and const.

Ans:

  1. var is global or functional scope. It can be reassigned. It can be redeclared.
  2. let is block scoped. It can be reassigned. It cannot be redeclared.
  3. const is vlock scoped. It cannot be reassigned. It cannot be redeclared. Have to assign value with declaration itself.
  
        console.log("---var---");
        var a=5;
        var a=10;
        console.log(a);      //5   //10

        console.log("---let---");
        let a=10;
        let a=20;      //we cannot redeclare.
        console.log(a);   //10    //error

        console.log("---const---");
        const a=10;
        a=20;
        console.log(a);  //throws error that assigned to fixed value (cannot be modified).

2) What is Hoisting?

Ans: It is deafult Javascript Mechanism when our variables and functions declarations are moved to top of their scope before the code execution.

        console.log("---Hoisting---");
        a=10;
        console.log(a);   //10
        var a;   // var a appears to move to the top of its scope.

3) What is Temperal Dead Zone?

Ans: The time between the variables hoisted and the variables get initialized is called the Temperal Dead Zone.

        console.log("---Temperal Dead Zone---");
        console.log(a);   //undefined
        console.log(b);   //Cannot access 'b' before initialization
        var a=10;
        let b=20;
